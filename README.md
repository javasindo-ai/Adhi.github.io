# CV. JAVAS INDONESIA KONSULTAN - Website

## Solusi Profesional untuk Lingkungan yang Berkelanjutan

Website company profile profesional untuk CV. JAVAS INDONESIA KONSULTAN - konsultan lingkungan terkemuka di Indonesia.

## 🚀 Teknologi

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework
- **Vanilla JavaScript** - No dependencies
- **AOS** - Animate on scroll library
- **Swiper** - Touch slider library
- **GitHub Pages** - Static site hosting

## 📁 Struktur Folder

```
.
├── index.html              # Beranda
├── about.html              # Tentang Kami

├── contact.html            # Kontak

│   ├── css/

│   │   └── tailwind.css    # Tailwind CSS
│   ├── js/
│   │   ├── main.js         # Main JavaScript
│   │   ├── navigation.js   # Navigation handler
│   │   └── form-handler.js # Form validation
│   ├── images/             # Image assets
│   ├── icons/              # SVG icons
│   └── data/               # JSON data

├── sitemap.xml
├── manifest.json
├── favicon.ico
└── README.md
```
